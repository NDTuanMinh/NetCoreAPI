Đây là đoạn văn được định dạng bằng CSS nội bộ, thông qua thẻ <style> trong phần <head>.

Đây là đoạn văn được định dạng bằng CSS nội tuyến (inline) ngay trong thẻ <p>.

Nếu sử dụng CSS ngoài (external CSS), bạn sẽ liên kết bằng thẻ:
<link rel="stylesheet" href="style.css">
(Ví dụ này chỉ mô phỏng vì không có file style.css kèm theo)
